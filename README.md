# Recipe Website
This is an assignment about making a website related to a cooking recipe. I chose **Char Siu Egg Fried Rice**.

### There are 2 quirks I added to this website:
1. A **sequential fade in effect** for both the two lists (and mass fade in effect for the whole website itself)
2. A **hover effect** for all of the ingredients, turning the cursor into an image of the ingredient.
